About The Arena

The Arena is an early internet themed competition site where coders attempt to complete a Python problem quickly. May the fastest win!

The Arena was written in Python3 with Django as the main driver for the site. Django Channels, an addon, was used for handling websocket connections.

To manage everything, we used Docker Compose. We have used database images (Redis, Postgres), an Nginx image, and a custom Python image that runs Django.
